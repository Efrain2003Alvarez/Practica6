¿Qué es HTML?

Es un lenguaje que se utiliza para la creación de paginas en la WWW

¿Qué es CSS?

Es un lenguaje de programación utilizado para definir y controlar el diseño, el formato y la presentación de páginas web

¿Qué es JAVASCRIPT?

Es un lenguaje de programación ampliamente utilizado en el desarrollo web para hacer que las páginas web sean interactivas y dinámicas
